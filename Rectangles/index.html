The calculator below can be used to estimate the maximum number of smaller rectangles - or squares - that may fit into a larger rectangle or square. The calculator can be used to calculate applicatons like

  • the number of bricks in a wall
  • the number of rectangular cuts possible from a steel or plywood plate 

Input the large rectangle inside dimensions - and the outside dimensions of the smaller rectangles.

Default values are for 0.5 x 0.8 inch rectangle inside a 10 inch x 10 inch square. The calculator is generic and all units can be used - as long as the same units are used for all values.

large rectangle width (in, mm, m)

large rectangle height (in, mm, m)

small rectangles width (in, mm, m)

small rectangles height (in, mm, m)

space between small rectangles  (in, mm, m)

Rectangular Pattern

 

Note! - the algorithm used for this tool is quite simple. Be aware that for some combinations of values more rectangles can be added by rearranging the layout.
